# 文件管理器前端项目

一个基于 Vue 3 的文件管理系统前端界面，支持文件/文件夹的查看、创建、删除、重命名、移动、上传、下载等操作。

## 功能特点
- 直观的文件列表展示（区分文件和文件夹）
- 面包屑导航，支持快速跳转目录
- 完整的文件操作：新建文件夹、上传文件、下载文件、查看文本内容等
- 登录状态管理与路由守卫

## 技术栈
- Vue 3（Composition API）
- Element Plus（UI 组件库）
- Pinia（状态管理）
- Vue Router（路由管理）
- Axios（网络请求）

## 项目结构
- `src/api/`：API 接口封装
- `src/router/`：路由配置
- `src/stores/`：状态管理
- `src/components/`：通用组件（部分冗余组件已标注可删除）

## 项目安装
1. 安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`
3. 构建生产版本：`npm run build`
